# 📦 倉庫庫存管理系統 - 快速使用指南

## 系統概述

**Sistem Gudang Inventory** 是一個現代化的倉庫物料進出管理系統，支援中文和印尼文雙語界面。

### 主要特色

✅ **QR Code 掃描** - 快速取貨/入貨  
✅ **購物車模式** - 一次批量操作多種商品  
✅ **分類管理** - 按商品分類快速篩選  
✅ **權限管理** - 員工和管理員分權限  
✅ **交易記錄** - 完整的進出庫歷史  
✅ **庫存預警** - 自動提醒低庫存商品  

---

## 🔐 登錄方式

### 一般員工
- **PIN 碼**: `0000`（臨時，由管理員分配）
- **進入後可以**: 取貨、申請、查看目錄

### 管理員
- **PIN 碼**: `123456`（預設，請更改）
- **進入後可以**: 管理商品、員工、採購、查看所有記錄

## 📱 介面與功能

### 1️⃣ **首頁 (HOME)**
- 📊 庫存概覽（總價值、低庫存提示）
- 🔔 近期申請列表
- 🔄 最近交易記錄

### 2️⃣ **取貨 (AMBIL/TAKE)**
使用情況：員工取出所需商品

**步驟：**
1. 🔍 搜尋商品或點選分類篩選
2. 📷 掃 QR Code 自動加入（或按 + 手動添加）
3. 🛒 調整數量
4. 👤 輸入你的名字
5. ✅ 確認取貨

**特色：**
- 支援大圖瀏覽商品
- 購物車模式：同時取多種商品
- 即時庫存警告

### 3️⃣ **入貨 (MASUK/RECEIVE)** - 僅管理員
使用情況：新商品入庫或補貨

**步驟：**
1. 搜尋商品
2. 掃 QR Code 或手動選擇
3. 輸入入庫數量
4. 確認入貨

### 4️⃣ **申請 (PERMINTAAN)**
使用情況：員工申請新商品

**步驟：**
1. 選擇現有商品 或 申請新品
2. 填寫申請內容
3. 留下備註（選填）
4. 送出申請

**狀態：**
- 📋 待審（等待管理員批准）
- ✅ 已批准（已核可）
- ❌ 已拒絕

### 5️⃣ **記錄 (RIWAYAT)** - 僅管理員
完整的進出庫交易歷史
- 取貨記錄
- 入貨記錄
- 操作人員和時間

### 6️⃣ **目錄 (CATALOG)**
所有商品一覽
- 按分類分組顯示
- 大圖瀏覽商品
- 顯示庫存數量

### 7️⃣ **管理 (ADMIN)** - 僅管理員

#### 📦 商品管理
- ➕ 新增商品（名稱、數量、最低庫存、分類、照片）
- ✏️ 編輯商品資訊
- 🗑️ 刪除商品
- 📸 拍照或上傳商品照片（自動壓縮）
- 🔖 生成 QR Code（打印並貼在商品上）

#### 💰 採購管理
- ➕ 新增採購單（供應商、價格、數量）
- 記錄採購歷史
- 計算平均成本

#### 👥 員工管理
- ➕ 新增員工
- 🔑 設定員工 PIN 碼
- 🗑️ 刪除員工

#### 🏷️ 分類管理
- ➕ 新增分類
- 🗑️ 刪除分類

#### 📊 Excel 匯入
- 上傳 Excel/CSV 批量導入商品
- 支援格式：名稱 | 數量 | 單位 | 最低庫存 | 採購網址 | 分類 | 條碼

#### 🔑 PIN 管理
- 生成臨時 PIN 碼給新員工
- 一鍵複製分享

## 🎯 常見工作流程

### 📋 日常補貨流程
1. 管理員檢查「首頁」低庫存提示
2. 進入「採購」新增採購單
3. 採購到貨後，進入「入貨」掃 QR Code 或搜尋確認
4. 輸入入庫數量 → 確認完成

### 🛍️ 員工取貨流程
1. 員工登入（PIN: 0000）
2. 進入「取貨」頁面
3. 📷 掃商品上的 QR Code（快速）或搜尋商品（手動）
4. 👤 輸入名字 → 確認取貨 ✅

### ⚙️ 新增商品流程（管理員）
1. 進入「管理」→ 「商品管理」
2. 點擊「新增商品」
3. 填寫：名稱、初始數量、單位、最低庫存、分類
4. 📷 拍照或上傳商品圖片
5. 📱 系統自動生成 QR Code
6. 打印 QR Code 標籤貼在商品上
7. 儲存完成

## 📞 常見問題

**Q: 照片可以多大？**  
A: 系統自動壓縮至 800px 以內、品質 72%，節省存儲空間。

**Q: QR Code 可以在哪裡用？**  
A: 打印並貼在實體商品或貨架上，員工用手機掃即可快速取貨。

**Q: 支持多少個用戶同時操作？**  
A: 取決於 Supabase 付費方案，基本方案支持足夠的並發。

**Q: 數據會備份嗎？**  
A: 所有數據存儲在 Supabase，自動備份，支持數據導出。
