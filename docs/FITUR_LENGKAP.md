# Gudang One — 完整功能總表 (Daftar Fitur Lengkap)

> 承接「10 亮點」那份簡報,這裡把**系統所有功能**列出來(不只員工端)。
> 每項標明:模組、功能、誰能用。給設計/文件參考用,可挑要放進手冊的部分。
>
> 角色代號:**S**=員工(Staff) · **G**=訪客(Guest) · **AG**=Admin Gudang 👑 · **AO**=Admin Office 🏢 · **SA**=Super Admin 🔐

---

## A. 登入與角色 (Login & Peran)
| 功能 | 說明 | 誰能用 |
|---|---|---|
| 員工 PIN 登入 | 選名字 + 4 位 PIN,後端驗證,免密碼免 email | S |
| 訪客登入 | 「Pengguna Baru」臨時碼,填名字即可用 | G |
| Admin Office 登入 | 6 位 PIN(OFFICE_PIN),協助設定員工/產品資料 | AO |
| Super Admin 登入 | 6 位 PIN(ADMIN_PIN),最高權限 | SA |
| 記住登入 | 同一台裝置記住 session,不用每次重登 | 全部 |
| 每人每倉權限 | 逐倉設定 Ambil/Masuk/Opname/Produksi/Kirim 開關 | 由 AO/SA 設 |

## B. 多倉庫 (Multi-Gudang)
| 功能 | 說明 | 誰能用 |
|---|---|---|
| 四個倉庫 | HARDWARE / SJA / DENIKIN / OLENTIA,庫存各自獨立 | 全部 |
| 切換倉庫 | 頂部切換器,秒切(背景預載) | 有多倉權限者 |
| 「Semua Gudang」總覽 | 管理員一次看所有倉,卡片標倉庫徽章 | AG/AO/SA |
| 倉庫篩選 chips | Kelola Barang 頂部一鍵聚焦某倉 | AG/AO/SA |
| 共用商品主檔 | 同名商品跨倉共用名稱/照片/單位,庫存獨立 | — |

## C. 首頁 (Beranda)
| 功能 | 說明 | 誰能用 |
|---|---|---|
| 庫存總覽清單 | 名稱/編號/規格/供應商 + 現量/低標,🟢🟠❌ 狀態 | 全部 |
| 即時搜尋 | 打字即篩選庫存 | 全部 |
| 低庫存區塊 | 依分類分組,可展開,快速看哪些要補 | 全部 |
| 到期預警 | 30 天內到期的批次列表(近效期排前) | 全部 |
| 快捷任務按鈕 | 一鍵進 Ambil / Masuk / Cek Stok | 依權限 |

## D. 領料 (Ambil) — 出庫
| 功能 | 說明 | 誰能用 |
|---|---|---|
| 購物車式領料 | 多樣加入,一次確認,stok 自動扣 | S(有 ambil 權限) |
| FIFO「Pakai dulu」 | 系統指出該先用的 lot(最舊/最近效期) | S |
| 庫存不足保護 | 提示剩餘量、不允許扣成負數 | — |
| 部分成功處理 | 批次扣料若部分失敗,回報「N 成功/M 失敗」 | — |
| 取消已領 | 撤銷領料,庫存還原 | AG/SA |

## E. 收料 (Masuk) — 入庫
| 功能 | 說明 | 誰能用 |
|---|---|---|
| 收貨入庫 | 選收貨人、品項、數量,stok 增加 | S(有 masuk 權限) |
| Lot / 效期登記 | No.Lot、生產日、到期日,建立可追溯批次 | S |
| 停留續收 | 存檔後留在頁面,連續收多筆 | — |

## F. 申請 (Minta) — 缺料/採購
| 功能 | 說明 | 誰能用 |
|---|---|---|
| 送出申請 | 選/自填品項、數量、附照片、備註 | 全部 |
| 🔍 線上找料 | 一鍵搜 Shopee/Tokopedia/BigGo(比價),手機直開 App | 全部 |
| 貼商店連結 | 找到後貼進備註,連結自動可點 | 全部 |
| 審批流程 | ⏳ 待審 → ✓ 核准(直接入庫)/ ✕ 拒絕 | AG/SA 審批 |
| FAMMS 進單 | 從 FAMMS 系統自動帶入採購申請 | 系統自動 |

## G. 盤點 (Cek Stok / Opname)

|---|---|---|

| 只存有改的 | 改過的欄位標黃,只存這些,避免誤動 | — |
| 掃描輔助 | 📷 相機 / barcode gun 掃碼定位 | — |
| 失敗保留重試 | 部分存檔失敗時保留輸入,可原地重存 | — |

## H. 品項管理 (Kelola Barang)
| 功能 | 說明 | 誰能用 |
|---|---|---|
| 新增/編輯品項 | 名稱、編號、規格、單位、照片、描述、供應商 | AG/AO/SA |
