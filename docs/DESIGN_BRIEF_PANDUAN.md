# Design Brief — Gudang One 員工使用手冊

> 給 AI / 設計師的功能簡報。**視覺風格自由發揮**,本文件只定義:目標、受眾、必須包含的內容與功能結構。文案已備好(印尼文),可直接使用。

---

## 1. 這是什麼

- **產品**:Gudang One — 多倉庫庫存管理系統(HARDWARE / SJA / DENIKIN / OLENTIA 四個倉)
- **交付物**:一份「員工使用手冊」,類似行銷說明書/入職手冊
- **用途**:新員工上手、貼在倉庫現場、發到員工手機
- **載體**:HTML 網頁為主,需可列印成紙本(A4)

## 2. 受眾(設計時最重要的限制)

- 印尼倉庫技師,**非專業電腦使用者**,部分年長/戴手套作業
- 閱讀語言:**印尼文**(手冊全文印尼文;系統介面本身也是印尼文)
- 使用裝置:手機為主、倉庫共用平板、辦公室電腦
- 設計含義:**大字級、高對比、步驟編號清楚、一眼能認出畫面上的按鈕**

## 3. 手冊必含的內容區塊(依序)

### 3.1 封面
- 產品名:📦 Gudang One
- 副標:Panduan Karyawan(員工手冊)
- 一句話定位:「Ambil barang, terima barang, minta barang — semua cukup dari HP kamu.」
- 版本日期、支援裝置(HP / Tablet / Komputer)

### 3.2 十大主打亮點(核心賣點,員工視角)

| # | 亮點 | 說明重點 |
