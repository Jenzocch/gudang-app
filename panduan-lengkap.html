我現在要做什麼？→ 按這個

找到你目前的狀況，點開直接看步驟——不用滑到下面找。（括號內為系統上的印尼文按鈕名稱）

🚚供應商送貨來了▶
  1. 選收貨人、物品與數量
  2. 填批號 (No. Lot) 與有效日期（包裝上有的話）
  3. 按「確認收料」
↓ 看完整說明：收料 Masuk
🔧要拿東西出來用▶
  1. 搜尋物品，用 −／+ 調數量放入購物車
  2. 優先拿標示「Pakai dulu 先用這個」的批次
  3. 按「確認領料」
↑ 看完整說明：領料 Ambil
😥東西沒了 / 需要採購▶
  1. 選名字，選現有物品或自打新物品
  2. 按「線上找料」開三平台比價，複製連結貼備註
  3. 送出，等待管理員核准（核准即自動入庫）
🙋 看完整說明：叫料 Minta
👀看到不錯的供應商，但還不用買▶
  1. 貼上網購分享文字，或手動填名稱/連結/備註
  2. 存起來——不算庫存，不影響任何庫存數字
  3. 之後用關鍵字隨時找回來；真的要買時一鍵「轉入庫存」
🗂️ 看完整說明：供應商 / 產品知識庫
📋被交代要盤點庫存▶
  1. 選一個分類（不用一次全盤）
  2. 數架上的量，打進正確數字（可掃描 / 條碼槍）
  3. 按「儲存」——只有改過（黃色）的欄位會存
🔢 看完整說明：盤點 Cek Stok
🤷不知道東西放哪個架子▶
  1. 在首頁或分頁裡搜尋找到這個商品
  2. 點商品旁的 📍 圖示
  3. 直接看貨架實拍照片，新人也能立刻找到
⚠️出現黃色 / 紅色警告▶
  1. 🟠 黃色＝快沒了；❌ 紅色／❌＝已經沒貨或快過期
  2. 需要補貨就走「叫料」，或直接聯絡平常的供應商
  3. 不是你的職責就可以先關掉——管理員也會同步收到通知
看叫料流程
🆕第一次使用這個系統▶
  1. 用手機瀏覽器打開系統網址
  2. 選你的名字，輸入 4 位數 PIN（新人選「Pengguna Baru」）
  3. 選你要進的倉庫，開始使用；預設 PIN 1234 請盡快改掉
🔑 看完整說明：登入
❓找不到某個功能分頁▶
  1. 手機上看不到某分頁＝你目前沒有那項權限，不是系統壞了
  2. 找倉庫管理員或辦公室管理員開通
  3. 開通後重新整理頁面即可看到

重點最常用、最有感的功能

重點功能

先掌握這幾個，日常九成的事都搞定了。

🔑

4 位數 PIN 登入

選名字、輸 PIN，免帳密免 email，手機記住不用每次登。

📤

購物車式領料

多樣加入一次確認，庫存自動扣、留下誰領的紀錄。

📋

FIFO「先用這個」

系統直接標出該先領的批次（最舊/最近效期），不用記。

📥

收料含批號效期

登記 Lot 與有效日期，食品級可追溯，撐起 FIFO。

🔍

叫料 + 線上找料

缺料一鍵搜 Shopee/Tokopedia/BigGo 比價，貼連結給管理員直接買。

🔢

手機盤點

免紙本，分類逐區清點，只存改過的數字。

📍

存放位置照片

點 📍 看貨架照片，新人也能立刻找到東西。

⚠️

自動預警

庫存快沒、快過期，開 App 自動彈窗提醒。

📱

手機 / 平板 / 電腦通用

版面自動適應，沒網路也看得到最後資料，可裝成 App。

供應商 / 產品知識庫（Katalog Supplier）

▶
  • 先記下來，不算庫存：逛到不錯的供應商/產品先存起來，不影響庫存量或低庫存警示管理
  • 關鍵字搜尋：找名稱、供應商、備註，以後要用隨時找回來管理
  • 貼連結自動記錄：跟品項表單同一套貼上功能，商品名/連結/價格自動帶入管理
  • 📦 一鍵「轉入庫存」：真的買了之後，一鍵把知識庫紀錄轉成正式庫存品項，不用重打管理，知識庫原紀錄保留不刪

操作最常做的五件事

日常操作步驟

🔑

A · 登入

所有同仁▶
  1. 用手機瀏覽器打開 gudang-one.denikin.co.id
  2. 選你的名字 → 輸入 4 位數 PIN
  3. 選你要進的倉庫（有多倉權限時）
新同仁？選「✏️ Pengguna Baru」。預設 PIN 1234，請盡快請管理員改掉。
📤

B · 領料（分頁 ↑ Ambil）

有領料權限▶
  1. 搜尋物品，用 − / + 調數量（放入購物車）
  2. 優先拿標示 📋「Pakai dulu 先用這個」的批次
  3. 按 ✓ Konfirmasi Ambil（確認領料）
庫存不夠？系統會提醒且不讓變負數。沒了 → 改用叫料。
📥

C · 收料（分頁 ↓ Masuk）

有收料權限▶
  1. 選收貨人、物品 + 數量
  2. 填 批號 (No. Lot) 與有效日期（包裝上有的話）
  3. 按 ✓ Konfirmasi Masuk（確認收料）
一次到很多種？存檔後留在原頁，接著登記下一項。批號/效期是食品追溯與 FIFO 關鍵。
🙋

D · 叫料（分頁 🙋 Minta）

所有同仁▶
  1. 選名字，選現有物品或自打「新物品」
  2. 按 🔍 Cari online → 開三平台 → 複製連結 → 貼到備註
  3. 需要加 📷 照片 → 按 Kirim（送出）
狀態：⏳ 等待 → ✓ 核准 / ✕ 拒絕。連結管理員可直接點開買。
🔢

E · 盤點（按鈕 🔢 Cek Stok）

有盤點權限▶
  1. 選一個分類（不用一次全盤）
  2. 數架上的量，打進正確數字（可用 📷 掃描 / 條碼槍）
  3. 按 ✓ Simpan（儲存）
只有改過（黃色）的欄位會存，不動其他數字。
權限誰能做什麼

權限由管理員「按每個人、每個倉庫」設定。手機看不到某分頁 = 你還沒那個權限，請找管理員。

該做 & 不該做

✅ 該做

領料當下就馬上輸入 · 遵照「先用這個」批次 · 每次收料都填有效日期 · 拿到帳號盡快改掉預設 PIN 1234

拿了東西卻不輸入 · 用別人的名字操作 · 購物車放著沒按確認 · 把你的 PIN 告訴任何人
